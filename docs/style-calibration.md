## Patterns We Are Intentionally Emulating
1. Reader-first framing
- Start with the felt problem in plain language.
- Focus on what improves in daily life, not just item features.

2. Scan-friendly layout
- Use short sections and numbered ideas/checklists.
- Keep each recommendation easy to execute in one sitting.

3. Budget + effort realism
- Mention budget tiers, installation risk, and time expectations.
- Avoid aspirational language that ignores constraints.

4. Soft conversion style
- Add internal links where they naturally help the reader decide next steps.
- Mention products as optional deeper resources, not mandatory CTA interruptions.

5. ICP-specific language
- Renter ICP: deposit-safe, reversible, no-drill options.
- Small-space ICP: daily flow, visual calm, counter control.
- Style-forward budget ICP: personality without clutter, high-impact low spend.

## Generator Changes Aligned To This
- Blog generator now writes benefit-first narrative + practical 5-step weekend path.
- Micro-guide generator now writes compact helper-style guides (45-90 minute format).
- Internal linking language is now subtle and contextual.
- Hard-sell closing copy removed from micro guides.
