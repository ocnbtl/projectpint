# Research Notes (Checked: 2026-02-23)

## Pinterest Creative / Posting Constraints
- Pinterest ad specs list a **2:3 aspect ratio** recommendation for standard Pins. Source: [Pinterest business ad specs](https://business.pinterest.com/en-GB/pinterest-ad-specs/)
- Pinterest supports bulk create/upload workflows via Ads Manager. Source: [Pinterest bulk create help](https://help.pinterest.com/en/business/article/create-ads-in-bulk)

## Disclosure / Compliance
- FTC emphasizes disclosures should be difficult to miss ("clear and conspicuous") and understandable to ordinary readers. Source: [FTC Endorsement Guides FAQ](https://www.ftc.gov/business-guidance/resources/ftcs-endorsement-guides-what-people-asking)
- FTC also states that if a significant minority of readers could misunderstand a disclosure, it is not clear and conspicuous enough. Source: [FTC blog](https://www.ftc.gov/business-guidance/blog/2023/09/ftc-updates-endorsement-guides-what-businesses-need-know)

## Automation Scheduling
- GitHub schedule workflows run on the latest commit on the default branch. Source: [GitHub Actions: events that trigger workflows](https://docs.github.com/en/actions/using-workflows/events-that-trigger-workflows#schedule)

## Low-Cost Overlay Automation
- ImageMagick `-annotate` can render text overlays directly onto images for automation-friendly batch workflows. Source: [ImageMagick annotate option](https://imagemagick.org/script/command-line-options.php#annotate)

## Notes
